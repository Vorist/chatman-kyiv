Положите сюда скриншоты отзывов с именами:

  review-1.jpg
  review-2.jpg
  review-3.jpg
  review-4.jpg
  review-5.jpg

Рекомендации:
  - Формат: JPG (или WebP для лучшего веса)
  - Размер: ~560×760 px (или пропорция 1:1.36)
  - Вес каждого: <150 КБ (прогнать через https://squoosh.app)
  - Скрыть личные данные (номера, имена) если скриншоты реальных переписок

Если файлов в папке нет — на лендинге вместо картинок будут плейсхолдеры
с подписью какой файл куда положить. Это специально так сделано для
удобства — чтобы сразу было видно что ещё не готово.

Чтобы добавить 6-й, 7-й скриншот и т.д.:
  1. Положи файл review-6.jpg в эту папку
  2. Открой index.html, найди секцию <!-- Дубль набора ... -->
  3. Скопируй один блок <div class="marquee-card">...</div>
     и замени review-X.jpg на review-6.jpg (в ОБА набора — первый и дубль!)
